# Decisiones abiertas para resolver durante la Fase 1

Estas preguntas no deben bloquear el inicio, pero deben cerrarse antes de implementar el bloque afectado.

## Cantidades

- ¿Se permiten cantidades exactas fraccionarias para `UNIT`?
- ¿Qué escala decimal máxima se acepta?
- ¿Cómo se expresa “usar un poco” en modo aproximado: transición fija o nivel objetivo?

## Lotes

- ¿V1 permite dividir una existencia?
- ¿V1 permite fusionar existencias equivalentes?
- ¿Se conserva origen al fusionar?

## Fechas

- ¿Se distingue “fecha estimada” con un tipo explícito?
- ¿Existe una política V1 para caducidad tras apertura?

## Lista de compra

- ¿La sugerencia por poca existencia se basa en umbral por producto o regla global?
- ¿Puede haber dos entradas activas del mismo producto con notas diferentes?

## Historial

- ¿Qué operaciones serán reversibles en V1?
- ¿Durante cuánto tiempo se permite deshacer desde UI?

## Resolución

Cada respuesta aceptada debe actualizar la RFC correspondiente antes de implementar.
