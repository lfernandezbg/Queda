# ADR-004 — Documentación viva y versionada

- Estado: ACCEPTED
- Fecha: 2026-07-15

## Decisión

Las decisiones de dominio se registran en RFC cortas y las decisiones técnicas en ADR.

## Reglas

- No se documenta funcionalidad hipotética con detalle.
- Código, tests y documentación cambian en el mismo commit.
- Una decisión reemplazada se marca `SUPERSEDED`.
- La documentación nunca sustituye a los tests.
